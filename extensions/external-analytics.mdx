## Extension Configuration

<Steps>
  <Step title="Step 1">
    Go to the **Marketplace** in **Admin Panel** and install **External Chatbot Analytics** extension after successfully purchasing it first.     
    <img src="/images/extensions/external-analytics/anal-1.png" />    
  </Step>
  <Step title="Step 2">
    After installation is completed, go to **AI Settings** page in Admin Panel and click on **Extensions** tab there. Click on **External Chatbot Analytics** there. 
    <img src="/images/extensions/external-analytics/anal-2.png" />  
  </Step>
   <Step title="Step 3">
   To control the settings of **External Chatbot Analytics** exension, return to **AI Settings** page in Admin Panel and click on the **Extensions** tab, and select **External Chatbot Analytics** extension.
    1. Turn on the checkboxes for **External Chatbot Analytics Feature** and **External Chatbot Analytics Free Tier Access** 
    <Note>If you don't turn on External Chatbot Analytics Free Tier Access, then users who did not subscribe to any subscription plan where you have External Chatbot Analytics feature enabled, will not have any access to this feature</Note>
    2. After you are done with configuration, click on the **Save** button.
    <img src="/images/extensions/external-analytics/anal-3.png" /> 
  </Step> 
  <Step title="Step 4">
  <Check>
 Congratulations! You now have completed the External Chatbot Analytics extension configuration. Proceed to the section below on how to use it.
</Check>
    
  </Step>
</Steps>

## How to Use

Using External Chatbot Analytics extension is very easy and straightforward. It gives user the full metics on his external chatbot usages and metrics including sentiment analysis. 

1. Go to **AI Chatbots** in User panel. 
2. If user's account has access to External Chatbot Analytics, he can see a new **Chatbot Analytics** button there.
<img src="/images/extensions/external-analytics/anal-4.png" /> 
3. Upon clicking on it, user will see all details of each of their external chatbots.
